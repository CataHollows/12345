Una página no puede ser manipulado de forma segura hasta que el documento está "listo". jQuery detecta este estado de preparación para usted. Código incluido dentro ready (documento) $ () sólo se ejecutará una vez que el m odelo de objetos de página de documento (DOM) está listo para el código JavaScript que se ejecutará. Código que se incluye dentro de $ .load (ventana) (function () {...}) se ejecutará una vez toda la página (imágenes o marcos flotantes), no sólo el DOM, está listo
